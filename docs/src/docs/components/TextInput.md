# Text Input
A component to get user input inside a text field.

## Usage
```tsx
<Input placeholder='Type some text...' />
```

## API Reference
| **Prop**   | **Type**  | **Default** |
|------------|-----------|-------------|
| `children` | ReactNode |             |
| `leftIcon` | ReactNode |             |
| `prefix`   | string    |             |
| `error`    | boolean   | false       |
